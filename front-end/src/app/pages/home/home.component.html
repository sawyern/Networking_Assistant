What is The In?

The In is an event planning application that helps attendees start conversations and connections. You will be able to set up a profile, set up an event, invite others, create notes, and keep track of connections that you have made. The In will help you analyze your interests, job experiences, and hobbies to help you start a conversation with others around you.
